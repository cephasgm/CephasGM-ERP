## What it is
A static, PWA-capable front-end for an ERP system that includes modules for Finance, HR, Inventory, Sales, Projects, and more.

## Quick start (local)
1. Clone the repo:
   `git clone https://github.com/cephasgm/CephasGM-ERP.git`
2. Serve it locally (simple HTTP server):
   `npx http-server . -p 8080`
3. Open `http://localhost:8080` in your browser.
